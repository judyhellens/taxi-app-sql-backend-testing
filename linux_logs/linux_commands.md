# Linux Commands Used

## Find requests from specific IPs

```bash
grep "233.201." logs.txt
```

---

## Create directories

```bash
mkdir bug1
mkdir events
```

---

## Filter HTTP 400 errors

```bash
grep "400" main.txt > 400.txt
```

---

## Filter HTTP 500 errors

```bash
grep "500" main.txt > 500.txt
```
