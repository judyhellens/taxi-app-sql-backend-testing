# Taxi Application — SQL & Backend Testing Project

## 📌 Project Overview
Technical QA project focused on SQL analysis, backend validation, Linux log investigation, debugging, and database testing for a taxi application platform.

---

## 🚀 Technologies & Tools
- SQL
- PostgreSQL
- Linux Console
- Backend Testing
- API Validation
- Log Analysis
- Debugging
- Data Validation

---

## ✅ Project Scope

### Linux Log Investigation
Performed backend log analysis using Linux commands to:

- Identify requests from specific IP addresses
- Investigate HTTP 400 and 500 errors
- Organize logs into structured reports
- Filter operational failures
- Validate backend behavior

---

### SQL Database Analysis
Developed SQL queries to:

- Analyze taxi company data
- Validate ride information
- Verify business logic
- Investigate weather condition impacts
- Detect inconsistencies in backend calculations

---

## ⚡ Key Achievements

- Improved backend investigation processes
- Validated database consistency
- Supported debugging and operational analysis
- Structured backend logs into organized reports
